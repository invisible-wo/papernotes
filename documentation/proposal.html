The PaperNotes Project

Description

The PaperNotes is an open source project and a fun platform for researchers and scholars to share comments, ideas and suggestions. Also, it is a social network to connect people who working in the same area closer.

Like most movie review website, PaperNotes is a social platform for scholars to provide their comments on papers or conferences. Each paper is like a movie and readers can rate it, write comments for it. Depends on individual data, PaperNotes also recommends new or classical papers. Users can add items and their descriptions to the web and under each item, users also can write review or comments.

Problems:

  • What is the representation for a paper on a page? Books have covers and movies have posters.
  • Relational Database?

Background

Review websites: Douban, Yelp, IMDb

Social Networks: Facebook, Academia.edu.

Proposed Solution

Web Crawler

Initially, information of paper descriptions will be got from web digital library and stored in database. Thus, we need to crawl paper information to generate a basic database.

Sources:

  • ACM Digital Library
  • IEEE Xplore
  • CiteSeerX

Database

MySQL database for back end of website.

Paper Descriptions:

  • ID
  • Type: (Article, Book, Booklet, Conference, Electronic, Inbook, Incollection, Inproceedings, Manual, Masterthesis, Misc, Patent, Periodical, Phdthesis, Proceedings, Standard, Techreport, Unpublisheds)
  • Authors
  • Title
  • Booktitle
  • ISBN
  • Location
  • Publicated Date
  • Publisher

User Profile:

  • ID
  • Username: not real name, can be changed
  • Sex
  • Occupation
  • Birth Date(Invisible)
  • Intro

Comments Table:

  • ID
  • ParentID: If this comment is a reply to another comment
  • UserID: wrote by which user
  • PaperID: under which paper
  • Text: content of comment
  • Timestamp: time
  • Deleted: if it is deleted by author

Front End

Simple and easy-to-use front.

Each item(paper) has an unique webpage with rating from users and comments below.

Each users has a profile webpage with brief introduction, followers, a list of papers s/he read, a list of comment ss/he wrote.

Social Network

Each user has a profile page. Users can follow other people(similar to twitter or douban).
